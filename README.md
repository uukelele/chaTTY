# chaTTY
chaTTY is a terminal-based chatting application for people who live in the terminal.

## Installation

```
$ gh repo clone uukelele/chaTTY
$ cd chaTTY
$ pip install -e . -U
```

## Updating

```
$ cd chaTTY
$ git pull
$ pip install -e . -U # This command isn't always necessary, but it sometimes is needed.
```

## Usage

```
$ chatty
$ # Or alternatively
$ python -m chatty
```
